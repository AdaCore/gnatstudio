Welcome to the seventh internal GPS beta

Version 0.9.2

2002/05/17

ACT Europe

Overview of the changes since version 0.9.1:
  • Shortcut provided to build the current file with the default project
  • Minor debugger bug fixes
  • It is now possible to copy text from the HTML window
  • The main files in the program wizard are now taken into account
  • Menu items File->Messages->Save As... & Load Contents... implemented
  • Menu Build->Run implemented
  • Automatic indentation now configurable through the preferences dialog
  • The search box now contains predefined regexps for the various languages
  • The buttons "Find" and "Next" have been merged in the search box
  • New contextual menu "find all local references" (9905-008)

Overview of the changes since version 0.9.0:
  • Better feedback in the search interface
  • Users can now add custom menus to launch their own executables (see doc)
  • Key bindings can be changed dynamically (see doc)
  • Selection is now taken into account when indenting lines (using Ctrl-Tab)
  • Improvements and bug fixes in the indentation engine
  • Default width of the line number area is now wide enough
  • Line numbers can now be enabled/disabled using the preferences
  • Crash related to the preferences editor fixed
  • Better behavior when jumping to a new source location
  • New accelerator to build the default main program easily
  • Better behavior of the MDI under Windows
  • Consistent use of the same file selector widget
  • Switch editor supports multiple selection
  • It is now possible to generate projects with relative path information
  • Project Properties editor has been redesigned and now provides access to all the project properties, instead of providing multiple contextual menu items, which was confusing
  • Project Properties now allow you to modify settings for a given set of projects and/or scenarii
  • Better support of VCS, in particular in the project wizard and properties
  • Better handling of untitled (new) files
  • Documentation updates
  • Various bug fixes and small improvements

At this stage, the following functionalities are implemented:
  • File Open/Save/Close
  • Basic source editing
  • Search
  • Source highlighting
  • Automatic indentation
  • Unlimited Undo/Redo
  • Cut/Copy/Paste
  • AUnit macros
  • Basic integration with Emacs and vi
  • Preferences
  • Source Navigation, including:
    • Goto Declaration
    • Goto Body
    • Goto Previous/Next Location (Unlimited)
    • List all references to an entity in the project
  • All project related operations, in particular:
    • Default project
    • Project explorer with multiple views:
      • A view Project/Directory/Source/Entities
      • A view File System/Directory/Source/Entities
    • Project Wizard (to create projects easily)
    • Project properties dialog with:
      • Name, Languages, Tools
      • VCS configuration
      • Add/Remove source directories
      • Set/Change Object & Exec Directories
      • Add/Remove main files
      • Switch Editor (per project or per file)
      • Naming Scheme
    • Add/Remove project dependencies
    • Creation/Use of configuration variables
    • Project Hierarchy Browser
  • Source dependency browser.
  • Call graph (what calls what, what is called by what)
  • Integrated debugging, providing all the functionalities of GVD
  • Basic CVS support, including:
    • Status of files
    • Check in
    • Check out
    • Edit log
    • Add/Remove file
    • View log
    • Visual Diff
  • Compile/Build
  • Check file syntax
  • Jump from error message to source editor
  • Visual Diff between Files
  • Automatic generation of body
  • Source Reformatting
  • Window manipulation
  • Integrated Help for GNAT tools
