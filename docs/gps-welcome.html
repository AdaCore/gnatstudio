Version 0.8.1

2002/04/12

ACT Europe

Welcome to the fifth GPS beta

This is the fifth beta version of GPS.

Here is an overview of the changes since version 0.8.0:

  • New menu File/Open From Project

Here is an overview of the changes since version 0.7.0:

  • The naming scheme editor now supports multi-language editing.
  • Several bug fixes and minor improvements.
  • Initial support for C/C++ navigation.
  • Memory leaks suppressed.
  • Speed improvements.
  • New command line switch -p for specifying the project to load.

Here are the functionalities that are implemented at this stage:

  • File Open/Save/Close
  • Basic source editing
  • Search
  • Source highlighting
  • Automatic indentation
  • Unlimited Undo/Redo
  • Cut/Copy/Paste
  • AUnit macros
  • Basic integration with Emacs and vi
  • Source Navigation, including:
    • Goto Declaration
    • Goto Body
    • Goto Previous/Next Location (Unlimited)
    • List all references to an entity in the project
  • All project related operations, in particular:
    • Default project
    • Project explorer with multiple views:
      • A view Project/Directory/Source/Entities
      • A view File System/Directory/Source/Entities
    • Project Wizard (to create projects easily)
    • Add/Remove project dependencies
    • Add/Remove source directories/files
    • Creation/Use of configuration variables
    • Set/Change Object Directory
    • Switch Editor (per project or per file)
    • Naming Scheme
    • Project Hierarchy Browser
    Full support for Ada is provided, some support for C/C++ as well.
  • Source dependency browser.
  • Call graph (what calls what, what is called by what)
  • Integrated debugging, providing all the functionalities of GVD
  • Basic CVS support, including:
    • Status of files
    • Check in
    • Check out
    • Edit log
    • Add/Remove file
    • View log
    • Visual Diff
  • Compile/Build
  • Check file syntax
  • Jump from error message to source editor
  • Visual Diff between Files
  • Automatic generation of body
  • Source Reformatting
  • Window manipulation
  • Integrated Help for GNAT tools

Many improvements are already on the TODO list, so we strongly suggest that you take a look at this file before submitting suggestions for enhancements.
