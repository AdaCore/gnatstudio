Welcome to the first Glide2 beta

This is the first beta version of Glide 2. Many functionnalities are still not implemented, but here is a list of things that should work fine:
  • File Open/Save/Close
  • Basic source editing
  • Source highlighting
  • Automatic indentation
  • Search/Replace
  • Cut/Copy/Paste
  • AUnit macros
  • Cross References
  • All project related operations, in particular:
    • Default project
    • Project Wizard (to create projects easily)
    • Add/Remove project dependencies
    • Add/Remove source directories/files
    • Creation/Use of configuration variables
    • Set/Change Object Directory
    • Switch Editor (per project or per file)
    • Naming Scheme
    • Project Hierarchy Browser
  • Source dependency browser
  • Integrated debugging
  • Basic CVS support, in particular:
    • Status of files
    • Check in
    • Check out
    • Edit log
    • Add/Remove file
    • View log
  • Compile/Build/Run
  • Jump from error message to source editor
  • Simple Visual Diff
  • Metrics:
    • Cyclomatic Complexity
    • Maximum nesting level
    • Number of statements
    • Number of types
    • Number of routines
    • Number of specification lines
    • Number of body lines
    • Number of lines
    • Number of blank lines
    • Number of code lines
    • Number of comment lines
    • Number of mixed code and comment lines
  • Automatic generation of body
  • Source Reformatting
  • Window manipulation
  • Integrated Help for GNAT tools
