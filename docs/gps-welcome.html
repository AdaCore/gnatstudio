GPS is a complete integrated development environment that gives access to a wide range of tools and integrates them smoothly, unleashing the full power of GNAT Professional.

This version is considered as beta software, meaning that you may encounter some missing features or instabilities. For this reason we will particularly welcome suggestions for possible improvements, as well as bug reports.

Online Documentation

GPS comes with a tutorial that gives an overview of the functionalities by going through a complete session.

The GPS manual is a work in progress. This is a preliminary version where most of the sections are not finished.

GPS Features

The following functionalities are implemented:
  • File Open/Save/Close
  • Basic source editing
  • Contextual search
  • Source highlighting
  • Automatic indentation
  • Unlimited Undo/Redo
  • Cut/Copy/Paste
  • Compile/Build/Run
  • Check file syntax
  • Jump from error message to source editor
  • Comparison between files
  • Automatic generation of body
  • Source pretty printing
  • Powerful window manipulations
  • Integrated help system
  • AUnit macros
  • Basic integration with Emacs and Vi
  • Preferences dialog
  • Customization
  • Intelligent source navigation, including:
    • Goto declaration
    • Goto body
    • Goto line
    • Goto file spec/body
    • Goto previous/next location (unlimited)
    • List all references to an entity in current file
    • List all references to an entity in the project
    • Two-ways source dependency browser
    • Two-ways call graph
  • All GNAT project related operations, in particular:
    • Default project
    • Project explorer with multiple views:
      • A Project/Directory/Source/Entity view
      • A File system/Directory/Source/Entity view
    • Project wizard (to create projects easily)
    • Project properties dialog with:
      • Name, Languages, Tools
      • VCS configuration
      • Add/Remove source directories
      • Set/Change object & exec directories
      • Add/Remove main files
      • Switch editor (per project or per file)
      • Naming scheme
    • Add/Remove project dependencies
    • Creation/Use of configuration variables
    • Project hierarchy browser
  • Integrated debugging, including in particular:
    • Run/Step/Next/Finish/Continue
    • Breakpoints (on line, function, exception, ...)
    • Attach to running processes
    • Core file debugging
    • Enhanced display of variables and complex structures
    • Dereference of pointers by a simple mouse click
    • Detection of aliases
    • Display of registers, local variables, arguments
    • Call stack
    • Assembly view
    • Task view
    • Memory view
  • Version control support for ClearCase and CVS, including:
    • Status of files
    • Check in
    • Check out
    • Edit log
    • Annotate files (CVS only)
    • Add/Remove file
    • Revision history
    • Comparison betwen revisions
